# map-mind-move

Claude Code skill implementing the Map-Mine-Move (MMM) three-phase framework — rapidly gain expertise, surface hidden risks, and produce an actionable plan for any unfamiliar domain.

## Install

```bash
cp -r . ~/.claude/skills/map-mind-move/
```

## Phases

| Phase | Goal |
|-------|------|
| **Map** | Landscape overview — key players, tools, and patterns in the domain |
| **Mine** | Inversion analysis — what goes wrong, what the experts know to avoid |
| **Move** | Actionable cheat sheet — the 20% that produces 80% of results |

## Trigger Phrases

- "map mine move"
- "apply the MMM framework"
- "I'm entering a domain I know nothing about"
- "help me avoid blind spots in this project"

## What's Included

- `SKILL.md` — full MMM framework with prompts, examples, and context-compounding mechanic
